sha257sum

like sha256sum but not quite!

a totally normal completely necessary cli tool — fully manual sha-256, then 35 sequential rounds of recursive salt-interleaved cryptographic torment.

how it works

  1. sha-256 hash the input
  2. reverse the last 8 characters of the hex digest
  3. convert to bytes and interleave with one of ten absurdly named salts
  4. repeat steps 1–3 for 35 rounds
  5. final sha-256 + reverse last 8 chars → output
