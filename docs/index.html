sha257sum how does it work
  1. initial compression: the input is crushed via standard sha256sum
  2. the reveresal: the last 8 hex characters are reversed
  3. the interleave: the modified hash is interleaved byte-by-byte with one of 10 salts
  4. the recursion: we repeat steps 1–3 for 35 rounds of cryptographic torment
  5. final seal: there is one last sha256sum followed by one final last-8-hex-chars reversal
